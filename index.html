HTML/CSS Cheatsheet

HTML Tables

Table Tags

Tag Name Description
<table> Table This is the wrapper element for HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing the actual table data.
<tr> Table Row The container for a table row.
<td> Table Data The container for representing a cells data.
<th> Table Head v2 This represents the heading for either a row or column and is best used on individual cells. Use of <th> is advantageous for people who use screen readers, SEO and better control when implementing CSS to a table.
<tfoot> Table Foot The set of rows defining the footer in a table. Browsers rarely treat the content of this element differently but is helpful when using CSS.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a <td> element should span. This is used within a container, not as data.
rownspan Row Span Defines how many rows a <td> element should span. This is used within a container, not as data.
